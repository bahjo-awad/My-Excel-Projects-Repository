# 📊 Excel Data Analysis Project – Data Technician Bootcamp

## 📌 Overview
This project was completed as part of a **Data Technician Bootcamp (Week 1 – Excel)** and demonstrates foundational data analysis skills using Microsoft Excel.  

The work focuses on analysing **Retail Sales** and **Bike Sales datasets** to extract insights, clean data, and present findings through visualisations and summaries.

---

## 🧠 Skills & Techniques Demonstrated

### 📈 Excel Functions
- `SUM` – total calculations (e.g., commissions, sales totals)
- `SUMIF` – conditional summation
- `AVERAGE` – calculating mean values
- `AVERAGEIF` – conditional averages
- `DATE`, `MONTH`, `YEAR` – handling and analysing date-based data
- `UNIQUE` – extracting distinct values from datasets
- `VLOOKUP` – retrieving related data across tables
- `SWITCH` – categorising data based on logical conditions (e.g., sales volume levels)

---

### 🔎 Data Analysis Techniques
- Filtering data (e.g., sorting age groups highest to lowest)
- Sorting datasets to identify trends and restore original structure
- Creating **calculated metrics** such as commission totals and averages  
- Identifying key business insights from transactional data  

Example insights:
- Adults contributed the highest bike sales
